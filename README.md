# Video explicacion grafos.

Método deleteArista 

Elimina una arista entre dos vértices especificados en un grafo. Si el grafo es no dirigido, también se elimina la arista inversa. Utiliza la estructura Set para almacenar y gestionar aristas relacionadas con cada vértice.

Método getAdyacentes

Recupera y muestra todas las aristas adyacentes a un vértice origen dado. Imprime cada arista encontrada, indicando el vértice de origen y destino de cada una.

Método deleteVertice

Elimina un vértice del grafo, así como todas las aristas que lo tienen como destino. En grafos no dirigidos, también elimina las aristas donde el vértice eliminado es el origen. Afecta a todas las listas de adyacencia en el grafo.

Método existeArista

Verifica si existe una arista directa entre dos vértices en un grafo. Devuelve true si la arista existe y false si no. Este método busca en el conjunto de adyacencias del vértice origen para comprobar si contiene al vértice destino.
